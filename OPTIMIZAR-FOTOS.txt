⚡ TUS FOTOS TARDAN EN CARGAR? OPTIMÍZALAS

📊 PROBLEMA ACTUAL:
Las fotos de WhatsApp son MUY PESADAS (2-5MB cada una)
Esto hace que tu sitio web cargue LENTO

✅ SOLUCIÓN: Optimizar antes de subir

🛠️ OPCIÓN 1: Usar TinyPNG (MÁS FÁCIL)

1. Ve a: https://tinypng.com
2. Arrastra tus fotos
3. Espera a que se compriman
4. Descarga las fotos optimizadas
5. Usa esas fotos en tu sitio

   RESULTADO: 70-80% más ligeras, misma calidad

🛠️ OPCIÓN 2: Usar ImageOptim (Mac)

1. Descarga: https://imageoptim.com
2. Arrastra tus fotos a la app
3. Automáticamente las optimiza
4. Usa las fotos optimizadas

🛠️ OPCIÓN 3: Redimensionar en Preview (Mac)

1. Abre foto en Preview
2. Tools → Adjust Size
3. Ancho: 1920px (mantén proporción)
4. Calidad: 70-80%
5. Guarda

🎯 TAMAÑOS RECOMENDADOS:

Para galería web:
  • Ancho: 1920px máximo
  • Alto: Proporcional
  • Formato: JPEG
  • Calidad: 70-80%
  • Peso: 200-500KB por foto

📱 TU SITUACIÓN ACTUAL:

WhatsApp Images originales:
  ❌ Peso: 2-5MB cada una
  ❌ Resolución: 3000-4000px
  ❌ Sin optimización

Después de optimizar:
  ✅ Peso: 200-400KB cada una
  ✅ Resolución: 1920px
  ✅ Carga 10x más rápida

⚙️ PASOS RÁPIDOS:

1. Selecciona todas tus fotos
2. Súbelas a TinyPNG.com
3. Descarga el ZIP optimizado
4. Reemplaza las fotos en footage/
5. Sube los cambios a GitHub

💡 TIPS IMPORTANTES:

✓ Optimiza ANTES de subir a GitHub
✓ Mantén copias originales en otro lado
✓ Usa JPEG para fotos (no PNG)
✓ Nombres sin espacios: foto-1.jpg
✓ Siempre comprime imágenes grandes

🔍 CÓMO SABER SI FOTO ESTÁ OPTIMIZADA:

Haz clic derecho → Get Info
Busca "Size"

  • > 1MB = ❌ Necesita optimización
  • 200-500KB = ✅ Perfecto
  • < 100KB = ⚠️ Puede perder calidad

🚀 HERRAMIENTAS ONLINE GRATIS:

• TinyPNG - https://tinypng.com (RECOMENDADO)
• Squoosh - https://squoosh.app
• CompressJPEG - https://compressjpeg.com
• ImageOptim - https://imageoptim.com (Mac app)

📊 IMPACTO EN VELOCIDAD:

Fotos sin optimizar (ahora):
  • 15 fotos x 3MB = 45MB total
  • Tiempo carga: 10-30 segundos
  • ❌ Muy lento

Fotos optimizadas:
  • 15 fotos x 300KB = 4.5MB total
  • Tiempo carga: 1-3 segundos
  • ✅ Rápido

🎨 CALIDAD vs TAMAÑO:

Calidad 90%: Excelente pero pesada (1MB+)
Calidad 80%: Muy buena y ligera (400KB) ← RECOMENDADO
Calidad 70%: Buena y muy ligera (250KB)
Calidad 60%: Aceptable (150KB)

🔄 PROCESO AUTOMATIZADO:

Para Mac (Automator):
1. Crea nuevo "Quick Action"
2. Agrega "Scale Images" → 1920px
3. Agrega "Change Type" → JPEG, Quality 80%
4. Guarda como "Optimize for Web"
5. Click derecho en fotos → Quick Actions → Optimize
